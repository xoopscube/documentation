## Design System Theming

The backend administration theme uses a modular CSS architecture based on CSS custom properties (variables). This architecture provides a flexible and customizable styling system for the backend environment, featuring support for both light and dark modes, responsive layouts, and component-based styling.

The system uses CSS media queries and data attributes to switch between themes, and utilizes JavaScript to save preferences to localStorage

1. `@media` (prefers-color-scheme: dark) for automatic dark mode detection
2. Data attributes `data-theme="dark"` or `data-theme="light"` for manual selection
3. Color preset classes `.amber`, `.blue`, `.forestgreen`, etc. for accent colors


Theme admin can be customized in several ways:

1. Theme Switching: Using data-theme="light" or data-theme="dark" on the root element
2. Color Scheme Customization: Applying classes (.amber, .blue, etc.) to the root element
3. Variable Overrides: Creating custom stylesheets that override specific CSS variables
4. Component Extension: Adding new components that follow the existing naming conventions


Example of theme class implementation:

```html
<!-- Light mode with amber accent -->
<html data-theme="light" class="amber">

<!-- Dark mode with indigo accent -->
<html data-theme="dark" class="indigo">
```

   Using flexible grid layouts with minmax() and auto-fit

2. Breakpoint-Based Media Queries:   
   Adjusting layouts at defined screen widths

### Performance Considerations

The framework incorporates several performance optimizations:

1. CSS Variables: Reduces redundancy and simplifies theme switching
2. Modular Architecture: Allows loading only necessary components
3. Minimized JavaScript: Uses compressed versions of jQuery and jQuery UI
4. Responsive Images: Using appropriate sizing and lazy loading techniques
5. Hardware Acceleration: Using transform and opacity for animations

Additionally, the framework includes utilities for improving accessibility, such as proper ARIA attributes and keyboard navigation support.
